 * Generate some CSS from your CSS preprocessor (SASS)
 * Being able to minify your CSS output
 * Knowledge of the following SASS features:
     * Variables
     * Mixins (Functions)
     * Nesting
     * Partials & Import
     * Extend/Inheritance
     * Operators (Math)
